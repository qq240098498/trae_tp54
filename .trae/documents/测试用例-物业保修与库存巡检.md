## 一、测试概述

### 1.1 测试目标
验证物业报修管理系统的核心功能，包括物业保修主流程和库存巡检功能，确保系统功能正确、流程完整、用户体验良好。

### 1.2 测试范围
- **物业保修主流程**：报修登记、工单管理、维修工工作台、工单详情、耗材登记、签字确认、满意度评价
- **库存巡检功能**：巡检计划管理、巡检任务执行、异常转报修、库存管理、进销存记录

### 1.3 用户角色
| 角色 | 说明 |
|------|------|
| 业主 | 提交报修申请、查看报修进度、签字确认完工、满意度评价 |
| 物业管理员 | 查看所有工单、派单、处理超时升级、管理巡检计划、管理库存 |
| 维修工 | 查看待接单列表、接单、更新到场/完工状态、登记耗材使用 |
| 巡检员 | 执行巡检任务、记录巡检结果、异常转报修 |
